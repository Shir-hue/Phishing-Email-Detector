About this project

This tool detects phishing emails using machine learning. The project is designed to be readable and security-focused — showing a verdict and the top contributing indicators.

Overview Run an analysis

What you can do

  • Paste an email message and receive a verdict: Phishing or Legitimate.
  • See a confidence score for the classifier output.
  • Review top indicators that support the decision.

Security note

  • Privacy first: Your email data is never sent to a backend server, stored, or tracked. Any saved history remains entirely on your account.
  • Open and transparent: The entire source code is fully open-source. Anyone can audit the logic to verify how text inputs are analyzed.
